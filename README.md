# DeliveryApp

## Description
DeliveryApp is a React Native application that allows users to:
- Select a product.
- Enter a pincode.
- Receive an estimated delivery date based on logistics provider data.

## Setup Instructions

### Prerequisites
Ensure you have `Node.js`, `npm`, and `React Native CLI` installed.
